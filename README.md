# openvpn-install-script
Script for automate install OpenVPN on Linux (Debian, Ubuntu) 

# Download
git clone  https://github.com/housekore/openvpn-install-script.git

#Enter in oflder
cd openvpn-install-script

#Change permissions
chmod +x openvpn-install.sh

#Execute script
./openvpn-install.sh
